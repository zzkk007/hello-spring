4、Spring 事务的本质
    
    Spring 的声明式事务本质上是通过 AOP 来增强了类的功能
    Spring 的 AOP 本质上就是为类做了一个代理
        看似在调用自己写的类，实际用的是增强后的代理类。
        
    
    事务的传播特性：
        REQUIRES_NEW vs NESTED
        
        REQUIRES_NEW:始终启动一个新事务，两个事务没有关联
        NESTED: 在原事务内启动一个内嵌事务
            两个事务有关联
            外部事务回滚，内部事务也回滚
            内嵌事务回滚，不会响应外面的事务
